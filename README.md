# AI-Driven Diabetes Lifestyle Recommendation Web Application

# Project Overview
This project is a web-based application designed to support self-care among diabetic patients by providing personalised lifestyle recommendations. The system focuses on diet, exercise, and general wellness advice based on user-provided health data. The aim is to improve patient engagement, autonomy, and awareness in managing diabetes.

# Features
- User input for basic health information (e.g. age, BMI, glucose level)
- Personalised lifestyle recommendations
- Rule-based recommendation engine with scope for future ML integration
- Separate views for patients and clinicians (basic prototype)
- Web-based interface

# Technologies Used
- Frontend: HTML, CSS, JavaScript
- Backend: Python (Flask)
- AI Logic: Python (rule-based, extendable to ML)
- Version Control: Git and GitHub
- Development Environment: Visual Studio Code
